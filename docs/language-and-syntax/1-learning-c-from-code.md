  // ‡πÄ‡∏£‡∏µ‡∏¢‡∏á ‡πÇ‡∏î‡∏¢‡πÄ‡∏•‡∏∑‡∏≠‡∏Å‡πÄ‡∏≠‡∏≤‡∏ï‡∏±‡∏ß‡πÄ‡∏•‡πá‡∏Å‡∏™‡∏∏‡∏î‡∏à‡∏≤‡∏Å [l..n] ‡∏°‡∏≤‡πÉ‡∏™‡πà‡∏ó‡∏µ‡πà l ‡πÅ‡∏•‡πâ‡∏ß‡∏Ç‡∏¢‡∏±‡∏ö l
